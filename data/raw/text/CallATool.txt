        <h1 id="quick-code-tutorial-call-a-tool">Quick code tutorial: Call a tool</h1><div id="lastModifiedDate"><span>Last Updated</span>: 2025-11-08</div>
        <p>You can call a tool with an agent programmatically with the watsonx.ai API and SDKs.</p>
        <p>This tutorial demonstrates calling a tool with a foundation model. Work with Python, Node.js, or Curl. This example uses the <code>mistralai/mistral-large</code> foundation model, which supports text generation and tool calling. The model is provided
          with a tool called <code>add</code> and a message with the question “What is 2 plus 4?“. The response of the model is a message that contains the tool that is called and the method of calling that tool.</p>
        <p><img src="/docs/en/SSYOK8/wsj/developer-hub/images/stopwatch.png" alt="alt=&quot;&quot;" height="20"> You can complete this tutorial in less than 5 minutes.</p>
