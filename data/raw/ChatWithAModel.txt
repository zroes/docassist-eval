        <h1 id="quick-code-tutorial-chat-with-a-model">Quick code tutorial: Chat with a model</h1><div id="lastModifiedDate"><span>Last Updated</span>: 2025-11-08</div>
        <p>You can interact with foundation models in a conversational format using the watsonx chat API or SDKs. You can view different types of messages, such as system prompts, user inputs, and responses from the foundation model, including follow-up
          questions and answers. You can use the chat API to interact with a foundation model in the same way as the chat mode in the Prompt Lab.</p>
        <p>This tutorial demonstrates a chat interaction discussing the distance between Paris and Bangalore. Work with Python, Node.js, or Curl.</p>
        <p><img src="/docs/en/SSYOK8/wsj/developer-hub/images/stopwatch.png" alt="alt=&quot;&quot;" height="20"> You can complete this tutorial in less than 5 minutes.</p>
